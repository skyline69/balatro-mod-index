# Azeroth Balatro

A Balatro mod that brings the world of **Warcraft** into your poker runs.

## Features
- **200+ Warcraft Jokers** with Race, Class, Faction, and Weapon attributes
- **Equipment system** — consumables that attach to jokers and scale with item level
- **Enemy jokers** — hostile jokers with kill conditions that appear at blinds
- **Quest system** — consumables with randomized conditions and rewards
- **Leveling system** — jokers gain levels and ilvl over the course of a run

## Requirements
- [Lovely](https://github.com/ethangreen-dev/lovely-injector/releases)
- [Steamodded](https://github.com/Steamodded/smods/releases)

## Installation
1. Download the zip from the repository
2. Create a folder named `Warcraft` in your Balatro mods folder
3. Extract the zip contents directly into that folder

Make sure the structure is `Mods/Warcraft/main.lua` and not `Mods/Warcraft/AzerothBalatro/main.lua`.
