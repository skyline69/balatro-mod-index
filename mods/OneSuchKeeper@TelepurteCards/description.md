## Telepurte Cards

# Features
- 4 Deck Skins
- 4 Joker Skins

# Notes
This mod does **NOT** require [Malverk](https://github.com/Eremel/Malverk), but it is compatible with it and will add itself as a texture pack if [Malverk](https://github.com/Eremel/Malverk) is also installed.
