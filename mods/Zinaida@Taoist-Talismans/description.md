# Taoist Talismans
A content expansion mod for Balatro that introduces **Talismans**, a new card modifier system that occupies the Seal slot and acts as a second enhancement for cards. Same enhancements will result in a powerful stacked effect!

## Features
### Talismans
* New modifier/consumable type that occupies the Seal slot
* Multiple talisman types with unique effects
* Stacked effects for upgraded interactions

## Requirements
* Balatro v1.0.1o Full
* Steamodded v1.0.0~BETA-1620a or newer
* Older versions of Steamodded may work, but it's recommended to play on the newest version
