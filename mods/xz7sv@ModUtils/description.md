# ModUtils

Adds modding/lua utilities

# Stuff included
- UTILS.require - SMODS.load_file, but the paths are relative
- UTILS.{Mousebind, Starthook} - Steamodded objects that do what the name says
- table/list utilities inspired by vim.tbl_* and vim.list_* from Neovim's runtime
- and some other miscelaneous functions...

_there is no documentation, reading the code explains the code_
