# Collab++
Simple mod which adds new fan-made face card skins inspired by other games.
Disclaimer: Some cards may contain very mild character spoilers for their respective titles.

## Content
V1.0... Platformer pack 1 
- (Hearts) Celeste:         Theo, Madeline and Badeline
- (Diamonds) Fez:           Gomez, Hexahedron and Owls
- (Spades) Hollow Knight:   The Knight, Hornet and Zote
- (Clubs) Pizza Tower:      Peppino, Gustavo and The Noise

## Installation
Requires `steammoded-1.0.0~ALPHA-1404b` or newer - [link](https://github.com/Steamodded/smods/wiki).

Get the latest stable version of CollabPlusPlus from the [Releases](https://github.com/RadadaV2/Balatro-CollabPlusPlus/releases), then download and extract the zip.
