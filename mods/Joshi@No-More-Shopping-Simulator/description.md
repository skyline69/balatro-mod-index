# No More Shopping Simulator

### A Balatro mod that reduces the amount of time spent endlessly rerolling for certain cards

- While in a shop, open your collection and press Ctrl+Shift+F while hovering an item (a joker/ a consumable/ or whatever else might spawn in shop) 

- After closing the collection the game should reroll until either your item has been found in shop, your money hits 0 or 1000 rerolls have been done.

- If you need to stop the rerolling currently the only way is to go back to the main menu.(might be changed refer to ingame settings)



It should work with any modded content. If you find anything that doesn't work let me know and I'll fix it.
