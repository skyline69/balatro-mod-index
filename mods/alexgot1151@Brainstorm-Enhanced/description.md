# Brainstorm-Enhanced
## Description
Brainstorm-Enhanced is a fork of [Brainstorm](https://github.com/OceanRamen/Brainstorm/) focused on working with every content mod out there by dynamic resolution of new content. It also optimizes the seed finding without the help of [Immolate](https://github.com/SpectralPack/Immolate)

### Default Keybinds
- Save state to slot 1-5: hold `Z` and press `1-5` (or hold `1-5` then press `Z`)
- Load state from slot 1-5: hold `X` and press `1-5` (or hold `1-5` then press `X`)
- Fast reroll: `Ctrl + T`
- Toggle auto-reroll: `Ctrl + A`
- Stop active auto-reroll search: `Ctrl + P`

### Features
- Save states
- Fast reroll
- Auto-reroll with configurable filters
- Extended game speed options
- Dynamic detection of new/custom tags,packs,vouchers and legendary jokers
